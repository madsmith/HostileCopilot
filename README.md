## Installation

### Prerequisites

- Python 3.12 or higher
- pip package manager

### Install from Source

```bash
# Clone the repository
git clone https://github.com/martin/HostileCoPilot.git
cd HostileCoPilot

# Install in development mode
pip install -e .

# Or install with development dependencies
pip install -e .[dev]
```
